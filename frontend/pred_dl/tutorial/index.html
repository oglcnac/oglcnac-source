Tutorial


Please select species: 'human' or 'mouse'

  • For better results, it is recommended to choose appropriate species from which the proteins are, since different models were developed for each species.
  • Prediction for proteins from other species is technically possible, but the results should be used with extra caution!

Paste the protein sequence(s) in fasta format

  • Either a single sequence (example for human / example for mouse) or a file containing multiple proteins sequences (example for human / example for mouse) is acceptable.
  • There is no limit in the length of a protein sequence (can be up to 8000 A.A.)
  • Besides the protein sequence, the header must be included into the fasta file.

Click the 'submit for prediction' button for prediction

  • For individual proteins, it usually takes several seconds to get the results.
  • For a fasta file containing multiple protein sequences, it may take longer.

View the prediction results

  • A prediction score is given for each S/T residue.
  • Sites with scores >0.5, >0.95, > 0.99 can be regarded as low confidence, medium confidence, and high confidence, respectively.
  • Results can be downloaded for further analysis if needed.
